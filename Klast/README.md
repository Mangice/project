Кластеризация и балансировка нагрузки

Задание 1
Запустите на своей виртуальной машине два простых сервера Python на разных портах
Установите и настройте HAProxy, воспользовавшись материалами к лекции по ссылке
Настройте балансировку Round-robin на 4-м уровне.
Отправьте на проверку конфигурационный файл haproxy и скриншоты, на которых видно перенаправление запросов на разные серверы при обращении к HAProxy.

Задание 2
Запустите на своей виртуальной машине три простых сервера Python на разных портах
Настройте балансировку Weighted Round Robin на 7-м уровне так, чтобы первый сервер имел вес 2, второй — 3, а третий — 4
HAproxy должен балансировать только тот HTTP-трафик, который адресован домену example.local
